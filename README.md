## 📌 Project Overview
Adi Portfolio is a personal portfolio website built to showcase my skills, projects, and experience as a web developer. The website provides a professional online presence where recruiters, clients, or collaborators can view my work and learn more about my technical background.

---

## ✨ Features
- Responsive portfolio website
- Modern and clean UI design
- About Me section
- Projects showcase
- Skills section
- Contact section
- Fully responsive for desktop and mobile devices

---

## 🛠️ Technologies Used
- TypeScript
- HTML
- CSS
- JavaScript
- React / Next.js
- Netlify (for deployment)

---

## 💡 Why TypeScript?
TypeScript is used in this project to improve code quality and maintainability. It adds static typing to JavaScript which helps catch errors during development and makes the code easier to scale for larger applications.
